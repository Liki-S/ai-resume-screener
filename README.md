# AI Resume Screener

An AI-powered web application that ranks resumes based on job descriptions using NLP and semantic similarity.

## Features
- Upload multiple resumes
- Extract skills automatically
- Rank candidates using AI
- Clean web interface

## Tech Stack
- Python
- Flask
- sentence-transformers
- PDF processing

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
